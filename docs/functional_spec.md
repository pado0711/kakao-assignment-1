- 세션은 7일 동안 유효하며 로그아웃 시 즉시 폐기한다.

## Todo

- 선택 날짜 안에서 전체·진행중·완료 상태를 필터링한다.

## 요일 반복

- 시작일과 월~일 중 하나 이상의 요일을 선택한다.
- 종료일은 선택 사항이며 시작일보다 빠를 수 없다.
- 수정·삭제 시 이번 일정 또는 전체 반복 범위를 선택한다.
- 이번 일정의 완료, 내용 변경, 삭제는 전체 반복 규칙을 바꾸지 않는다.
